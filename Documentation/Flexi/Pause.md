## Declaration
```cs
public static void Pause(int id)
```

## Parameters
| Parameter | Type | Description |
| - | - | - |
| id | int | The ID of the interpolation to pause |

## Description
Searches for the interpolation with an ID of `id` and pauses it if it is not already paused
